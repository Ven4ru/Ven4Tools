## Каталог приложений

- **70+ приложений** в 10 категориях: браузеры, коммуникации, разработка, мультимедиа, утилиты и другие
- **Умная установка**: сначала winget, при недоступности — прямая ссылка
- **Приоритет источников установки**: настройте порядок winget → Chocolatey → Scoop → прямая ссылка — глобально или отдельно для каждой категории приложений
- **Проверка доступности** каждого источника — зелёный/красный индикатор в реальном времени
- **Альтернативные источники** для приложений, недоступных в регионе
- **Выбор диска установки** с отображением свободного места
- **Проверка целостности** — SHA256-верификация скачанного файла перед установкой
- **Определение установленных приложений** — уже установленные помечаются автоматически
- **Закреплённые приложения**: до 6 любимых программ выносятся отдельной полосой быстрого доступа над каталогом — установка в один клик
- **Профили каталога**: базовый / расширенный / полный — показывает только нужные приложения
- **Сортировка и фильтры**: скрыть установленные, компактный режим
- **Офлайн-режим и кэш каталога**: каталог загружается по схеме онлайн → локальный кэш → встроенная копия, поэтому программа открывается даже без интернета. В настройках можно включить принудительный офлайн-режим и указать папку для кэширования установщиков

### Поиск и свои приложения

- **Поиск по всем источникам**: если приложения нет в каталоге — под строкой поиска сразу появляются совпадения из трёх источников — winget, Chocolatey и Scoop, сгруппированные по разделам. Выбрал → приложение добавилось в «Пользовательские»
- **Добавление своих приложений** вручную: название, winget ID или прямая ссылка
- Пользовательские приложения **синхронизируются с аккаунтом** — доступны на любом устройстве после входа

### Пресеты

- **Сохрани набор приложений** как именованный пресет — один клик и весь список установится заново
- **Поделись пресетом** по короткому коду: другой пользователь вводит код и получает твой набор
- **Экспорт и импорт списка** выбранных приложений в JSON-файл — перенос набора между компьютерами или передача другому человеку, работает даже без аккаунта
- Пресеты **хранятся в облаке** и доступны с любого устройства после входа в аккаунт

### Избранное

- Отметь любое приложение звёздочкой ★ — включи фильтр «только избранные»

- **Настраиваемые таймауты** загрузки — применяются без перезапуска
- **Автозагрузка** при старте Windows и **уведомления** об обновлениях

## Лаунчер

Лаунчер — отдельная устанавливаемая программа, которая управляет клиентом Ven4Tools:

- **Установщик с мастером**: устанавливается в `%LOCALAPPDATA%\Ven4Tools\Launcher\`, регистрируется в «Программы и компоненты», появляется в меню «Пуск»
- **Чистый деинсталлятор**: полное удаление через «Программы и компоненты» или меню «Пуск»
- **Скачивание и обновление клиента** — выбор версии из списка, загрузка с GitHub с проверкой целостности, откат на предыдущую версию при ошибке
- **Самообновление**: лаунчер сам проверяет GitHub и устанавливает собственные обновления без UAC-запроса
- **Сворачивание в трей** — работает в фоне, всплывающие уведомления о новых версиях клиента
- **Автозапуск с Windows** без запроса прав администратора и **тихий старт** — запуск сразу свёрнутым в трей
- **Проверка компонентов**: winget, .NET Runtime — с предложением установить недостающее
- **История версий**: можно выбрать и запустить любую предыдущую версию клиента
- **Защита URL**: скачивание только с доверенных доменов GitHub

## Диагностика сети

Вкладка **«Сеть»** позволяет проверить состояние сетевого подключения:

- **Сетевые адаптеры** — список активных интерфейсов с IP-адресами
- **Пинг** — проверка доступности ключевых хостов (Google, Cloudflare, Яндекс, 8.8.8.8)
- **Доступность сервисов** — HTTP-проверка GitHub, Google, Яндекс, Steam, Discord
- **Внешний IP** — определение публичного IP-адреса
- **DNS** — проверка разрешения доменных имён
- **Инструменты** — сброс сетевых настроек Windows (netsh winsock reset и др.)

## Установка

### Вариант 1: Лаунчер (рекомендуется)

1. Скачайте `Ven4Tools.Setup-2.0.0.exe` из [релиза лаунчера](https://github.com/Ven4ru/Ven4Tools/releases/tag/launcher-v2.0.0)
2. Запустите установщик — никаких прав администратора не требуется
3. После установки откройте лаунчер из меню «Пуск» или с рабочего стола
4. Нажмите **«Загрузить Ven4Tools»** — клиент скачается автоматически
5. Нажмите **«Запустить Ven4Tools»**

При выходе новой версии лаунчер уведомит через трей — обновление в один клик.  
Удалить лаунчер можно через «Программы и компоненты» или меню «Пуск».

### Вариант 2: Клиент напрямую

1. Скачайте `Ven4Tools-Client-X.Y.Z.zip` из [последнего релиза](https://github.com/Ven4ru/Ven4Tools/releases/latest)
2. Распакуйте в любую папку
3. Запустите `Ven4Tools.exe` с правами администратора
