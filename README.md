# 🚀 Ven4Tools — Умный установщик программ для Windows

[![GitHub release](https://img.shields.io/github/v/release/Ven4ru/Ven4Tools)](https://github.com/Ven4ru/Ven4Tools/releases/latest)
[![Downloads](https://img.shields.io/github/downloads/Ven4ru/Ven4Tools/total)](https://github.com/Ven4ru/Ven4Tools/releases)
[![License](https://img.shields.io/github/license/Ven4ru/Ven4Tools)](LICENSE)

**Ven4Tools** — это современный инструмент для автоматической установки программ на Windows.  
Скачиваешь один раз, устанавливаешь всё, что нужно, и забываешь о ручной возне.

---

## ✨ Что умеет

| Возможность | Описание |
|-------------|----------|
| **📦 100+ приложений** | Браузеры, офис, мессенджеры, графика, разработка, игры и драйверы |

| **🎨 Современный интерфейс** | Тёмная тема, адаптивная вёрстка, цветные логи |
| **🔧 Системные настройки** | Автозагрузка, смена темы, настройка таймаутов |
| **📄 Установка Office** | Любая версия + любой язык |
| **🔑 Активация** | Интерактивный MAS + Online KMS |
| **🔄 Автообновления** | Лаунчер сам скачивает и устанавливает новые версии |

## 📦 Как это работает

**Пользователь**  
│  
├── Скачивает **Ven4Tools.Launcher.exe** (~5 МБ)  
│  
▼  

**Лаунчер**  
• Проверяет последнюю версию на GitHub  
• Скачивает **client.zip** (~50 МБ)  
• Распаковывает в выбранную папку  
• Запускает клиент  

▼  

**Клиент (Ven4Tools.exe)**  
• Загружает каталог приложений  
• Проверяет доступность  
• Устанавливает выбранное

## 📥 Установка

### Вариант 1: Лаунчер (рекомендуется)
1. Скачай `Ven4Tools.Launcher.exe` с [последнего релиза](https://github.com/Ven4ru/Ven4Tools/releases/latest)
2. Запусти от имени администратора
3. Нажми «Выбрать папку» и укажи место (например, `D:\`)
4. Лаунчер сам создаст папку `Ven4Tools` и установит программу
5. Нажми «Установить» — и жди, пока всё подготовится

### Вариант 2: Портативная версия
1. Скачай `client.zip`
2. Распакуй в любую папку (можно создать новую, например `C:\Ven4Tools`)
3. Запусти `Ven4Tools.exe` от имени администратора
