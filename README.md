## 🚀 Что нового в версии 2.2.4

### ✨ Новая архитектура
- **Лаунчер** — отдельное приложение, отвечающее за обновления
- Основная программа полностью очищена от кода обновлений
- Автоматическая проверка обновлений через GitHub API
- Умная установка: лаунчер сам скачивает программу, если её нет

### 🔧 Исправления
- Исправлена проблема с запуском от администратора
- Оптимизирован процесс установки
- Улучшена работа с правами доступа
- Добавлена иконка в установщик и ярлыки

## 📦 Установка

1. Скачайте установщик по [ссылке](https://github.com/Ven4ru/Ven4Tools/releases/download/v2.2.4/Ven4Tools_Setup_2.2.4.exe)
2. Запустите файл (потребуются права администратора)
3. Следуйте инструкциям на экране
4. После установки запустите программу через ярлык на рабочем столе
