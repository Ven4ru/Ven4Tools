# Ven4Tools — Умный установщик программ для Windows

[![GitHub release](https://img.shields.io/github/v/release/Ven4ru/Ven4Tools)](https://github.com/Ven4ru/Ven4Tools/releases/latest)
[![Downloads](https://img.shields.io/github/downloads/Ven4ru/Ven4Tools/total)](https://github.com/Ven4ru/Ven4Tools/releases)
[![License](https://img.shields.io/github/license/Ven4ru/Ven4Tools)](LICENSE)

**Ven4Tools** — бесплатная программа с открытым исходным кодом для автоматической установки популярного софта на Windows. Работает сразу после установки системы — **не требует установленного .NET** или других компонентов.

## 📦 Каталог приложений

- **70+ приложений** в 10 категориях: браузеры, коммуникации, разработка, мультимедиа, утилиты и другие
- **Умная установка**: сначала winget, при недоступности — прямая ссылка
- **Проверка доступности** каждого источника — зелёный/красный индикатор в реальном времени
- **Альтернативные источники** для приложений, недоступных в регионе
- **Выбор диска установки** с отображением свободного места
- **Проверка целостности** — SHA256 верификация скачанного файла перед установкой
- **Определение установленных приложений** — уже установленные помечаются автоматически
- **Профили каталога**: базовый / расширенный / полный — показывает только нужные приложения
- **Сортировка и фильтры**: скрыть установленные, компактный режим

### Поиск и свои приложения

- **Поиск с подсказками из winget**: если приложения нет в каталоге — под строкой поиска сразу появляются совпадения из winget. Выбрал → приложение добавилось в «Пользовательские»
- **Добавление своих приложений** вручную: название, winget ID или прямая ссылка
- **Удаление** пользовательских приложений
- Пользовательские приложения **синхронизируются с аккаунтом** — доступны на любом устройстве после входа

### Избранное

- Отметь любое приложение звёздочкой ★
- Включи фильтр «только избранные» — каталог покажет только нужное

## 👤 Аккаунты

- **Вход по email и паролю** или через **Яндекс OAuth**
- Пользовательские приложения и избранное **синхронизируются между устройствами** через аккаунт
- Полный контроль приватности:
  - анонимная статистика установок — отключаемо
  - история установок — отключаемо
  - локальное хранение данных без синхронизации — включаемо

## ⚙️ Система

- **Intel Turbo Boost** — включение и отключение без перезагрузки
- **Обновление установленных приложений** через winget: показывает список устаревших, обновляет одним кликом
- **Управление логами**: открыть папку с логами, просмотреть последний, очистить
- **Настраиваемые таймауты** загрузки — применяются на лету, без перезапуска
- **Автозагрузка** при старте Windows
- **Уведомления** о доступных обновлениях

## 🚀 Лаунчер

Лаунчер — отдельная программа, которая управляет клиентом Ven4Tools:

- **Скачивание и обновление клиента** — выбор версии, загрузка с GitHub
- **Самообновление**: лаунчер сам проверяет и устанавливает собственные обновления
- **Фоновая проверка обновлений** клиента и лаунчера
- **Сворачивание в трей** — работает в фоне, всплывающие уведомления о новых версиях
- **Автозапуск с Windows** — опционально
- **Тихий старт** — запуск сразу свёрнутым в трей
- **Проверка компонентов**: winget, .NET Runtime, права администратора — с предложением установить недостающее
- **История версий**: можно выбрать и запустить любую предыдущую версию клиента

## 📥 Установка

### Вариант 1: Лаунчер (рекомендуется)

1. Скачайте `Ven4Tools.Launcher.exe` из [последнего релиза](https://github.com/Ven4ru/Ven4Tools/releases/latest)
2. Запустите — лаунчер проверит компоненты и предложит установить winget если нужно
3. Нажмите **«Загрузить Ven4Tools»** — клиент скачается автоматически
4. Нажмите **«Запустить Ven4Tools»**

При выходе новой версии лаунчер уведомит через трей — обновление в один клик.

### Вариант 2: Клиент напрямую

1. Скачайте `Ven4Tools-3.0.0.zip` из [релиза](https://github.com/Ven4ru/Ven4Tools/releases/latest)
2. Распакуйте в любую папку
3. Запустите `Ven4Tools.exe` с правами администратора

## 🌐 Интеграция: Zapret

Во вкладке **«Сеть»** доступна интеграция [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) от [Flowseal](https://github.com/Flowseal) — инструмент для обхода DPI.

Ven4Tools **только** скачивает и распаковывает файлы zapret. Все настройки пользователь выполняет самостоятельно через оригинальное меню `service.bat`. Zapret использует системный драйвер WinDivert — **используйте на свой страх и риск** и ознакомьтесь с [документацией проекта](https://github.com/Flowseal/zapret-discord-youtube).

---

## 🔑 Интеграция: Microsoft Activation Scripts

Вкладка **«Активация»** предоставляет интерфейс для запуска [Microsoft Activation Scripts (MAS)](https://github.com/massgravel/Microsoft-Activation-Scripts) от [massgravel](https://github.com/massgravel).

Ven4Tools **не содержит** скрипты активации — программа только предоставляет удобный интерфейс для их запуска. Все действия выполняются оригинальными скриптами MAS. Пользователь несёт полную ответственность за использование инструментов активации.
