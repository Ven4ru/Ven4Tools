# Ven4Tools — Roadmap

> Актуально на: июнь 2026  
> Текущая версия: **3.4.1** (Latest, stable)  
> В разработке: **3.4.2**  
> Целевая версия: **4.0.0**

---

## 3.1.x — Управление установленным ✅ Выпущено

- InstalledTab — список через `winget list`, обновление, удаление
- HistoryTab — история установок, переустановка одним кликом
- Точка восстановления (WMI) перед массовыми операциями
- Экспорт / Импорт `.winget`
- Crash Reporting, Watchdog, push-уведомления
- Quick pins strip — до 6 закреплённых приложений

---

## 3.2.0 — Multi-source и офлайн ✅ Выпущено

*Установка из любого источника, работа без интернета*

- Chocolatey и Scoop наравне с winget и прямыми ссылками
- SourceOrderService — приоритет источников
- OfflineService — кэширование установщиков локально
- LocalInstallerDialog — добавление своих `.exe`/`.msi`
- Логирование установок по сессиям
- ConnectivityMonitor
- DebloaterTab

---

## 3.3.x — Teal UI + SplashWindow + Стабильность ✅ Выпущено

*Визуальный рестайл, стабильность, рефакторинг кода*

- Teal UI — тёмная navy/teal палитра, именованные стили
- SplashWindow — предзагрузка каталога при старте, проверка компонентов
- Launcher — самообновление, история версий, тихий старт в трей
- AppLogger — файловый лог с ротацией
- WingetRunner — унифицированный запуск winget с CancellationToken
- InstalledTab — «Обновить всё», сортировка, фильтр обновлений
- Рефакторинг: partial class split (CatalogTab → 7 файлов, Launcher → 6 файлов)

---

## 3.4.0 — Пресеты + Облачные аккаунты ✅ Выпущено

*Одна фича, которая заставляет возвращаться и рекомендовать другим*

- Сохранение набора приложений как именованного пресета
- Применение пресета одним кликом — всё устанавливается автоматически
- Поделиться пресетом по 4-байтному коду (`ven4tools.ru/p/XXXX`)
- Пресеты синхронизируются с облаком и доступны с любого устройства
- Bearer-токен авторизация, DPAPI-шифрование сессии
- Яндекс OAuth через WebView2

---

## 3.4.1 — Безопасность и стабильность ✅ Выпущено (2026-06-10)

*14 исправлений по результатам глубокого ревью*

- PAT вынесен с клиента на серверный прокси
- DPAPI шифрование токена в `session.json`
- ApiConfig — единый BaseUrl для всех сервисов
- WingetRunner — исправление кодировки кириллицы (WinHTTP на русской Windows)
- Утечки памяти: отписка от static events в `OnClosed`
- Все пустые `catch` → AppLogger

---

## 3.4.2 — Редактирование пресетов 🔄 В разработке

*Полноценное управление сохранёнными наборами*

- Изменить имя и описание пресета
- Добавить / убрать приложения из существующего пресета
- Обновить пресет на сервере (PATCH endpoint)

---

## 3.5.0 — Диагностика

*Утилита работает в обе стороны: не только устанавливает, но и чистит*

- Сканирование bloatware: список приложений, которые стоит удалить
- Автозагрузка: просмотр и отключение лишних записей
- Дубли и мусор в PATH
- Одна кнопка «Очистить очевидное» — применить все безопасные рекомендации
- Отчёт «До / После» с итогом по освобождённым ресурсам

---

## 4.0.0 — Portable и мультимашинность

*Логичная финальная точка для утилиты этого класса*

- Portable-режим: `.exe` без установки, можно запустить с флешки
- Синхронизация пресетов и избранного между машинами через аккаунт
- «Настроить с нуля»: прийти к другу, запустить с флешки, применить пресет — 5 минут и система готова
- Экспорт всего профиля (пресеты + настройки + история) в один файл

---

## Что не войдёт

| Идея | Причина отказа |
|---|---|
| Геймификация, достижения, медали | Мотивирует разработчика, не пользователя |
| Лидерборд, публичный профиль | Другой продукт, другая команда |
| Windows Image Edition | Отдельный проект уровня DISM-обёртки |
| VenRemote relay + монетизация | Слишком высокая инфраструктурная стоимость для текущего масштаба |
| MiniWindow (Ctrl+M) | Низкий приоритет, решается системным поиском |
